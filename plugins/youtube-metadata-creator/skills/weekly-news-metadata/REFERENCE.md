## 📚 参考資料一覧

### 1. タイトル戦略

詳細は `references/title-examples.md` を参照してください。

#### タイトルの5つのパターン
1. **時短強調型**: 「5分で」「3分で」を前面に出す
2. **網羅性強調型**: 「完全版」「全部」で安心感を提供
3. **損失回避型**: 「見逃せない」「知らないと乗り遅れる」
4. **トピック強調型**: 注目ニュースを先頭に配置
5. **問題解決型**: 視聴者の課題を明示して解決策を提示

#### 効果的な要素
- **数字**: 「5分で」「全7本」「2025年10月第4週」
- **具体的な企業・製品名**: OpenAI、ChatGPT、GitHub
- **行動を促す言葉**: 「キャッチアップ」「まとめ」
- **緊急性・重要性**: 「今週の」「最新」「見逃せない」
- **ターゲット明示**: 「忙しい人のための」「エンジニアが」

#### タイトル文字数
- **推奨**: 60-70文字（情報量と可読性のバランスが良い）
- **最小**: 40-50文字（簡潔版）
- **最大**: 80文字（情報量重視）

### 2. ハッシュタグ戦略

詳細は `references/hashtag-strategy.md` を参照してください。

#### ハッシュタグの3層構造

**第1層: 汎用系タグ（必須3-4個）**
```
#テックニュース #AIニュース #エンジニア #週刊ダイジェスト
```

**第2層: トピック系タグ（可変5-8個）**
- AIモデル関連: `#ChatGPT #Claude #Gemini #LLM #生成AI`
- 開発ツール関連: `#GitHub #VSCode #Cursor #GitHubCopilot`

**第3層: 視聴者属性系タグ（2-4個）**
```
#駆け出しエンジニアと繋がりたい
#エンジニアと繋がりたい
#フルスタックエンジニア
```

#### 推奨数
- **合計**: 10-15個
- **最小**: 8個（必須タグのみ）
- **最大**: 20個（情報過多にならない範囲）

## 💬 想定コメント・返信戦略

### 基本方針

動画公開後のエンゲージメント向上のため、視聴者からありそうなコメント5つとそれぞれへの返信を生成します。

#### コメント生成のポイント
- **自然な言葉遣い**: 実際の視聴者が書きそうなリアルな表現
- **多様な視点**: 驚き、質問、感謝、実体験の共有など
- **適度な長さ**: 2-5行程度が自然
- **具体性**: 動画内容や特定のタイムスタンプに言及

#### 返信のポイント
- **親しみやすさ**: 「ですます調」で丁寧かつフレンドリーに
- **感謝の表現**: 視聴やコメントへの感謝を必ず含める
- **追加価値の提供**: さらなる情報やアドバイスを添える
- **対話の促進**: 質問を返したり、次回へつなげる内容

### 週刊ニュース向けコメント5パターン

1. **ニュースへの驚きのコメント**
2. **週刊動画への感謝コメント**
3. **特定トピックへの質問コメント**
4. **実務への活用コメント**
5. **次回への期待コメント**
