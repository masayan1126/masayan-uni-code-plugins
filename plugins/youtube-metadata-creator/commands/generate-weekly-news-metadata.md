description: Generate YouTube metadata for weekly tech news videos
---

Execute the skill at `skills/weekly-news-metadata` to generate SEO-optimized YouTube metadata for weekly tech news digest videos.

This skill generates:
- 3 title patterns (time-saving, comprehensive, topic-focused)
- Description with timestamps and news links
- Trending hashtags for discoverability
- Expected viewer comments and replies

Usage: Run this command and provide the target week (e.g., "2025年1月第1週", "1/6-1/12").
