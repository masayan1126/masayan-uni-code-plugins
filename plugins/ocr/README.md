# OCR Plugin

Extract text from images using OCR and copy to clipboard.

## Features

- Extract text from PNG, JPEG, and other image formats
- Automatic clipboard copy
- Multi-language support (Japanese, English, etc.)

## Usage

```
/path/to/image.png を文字起こしして
```

Or use keywords like "OCR" or "文字起こし".

## Requirements

- macOS (uses `pbcopy` for clipboard)
