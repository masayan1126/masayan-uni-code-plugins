# OCR Skill - リファレンス

## 概要

画像ファイル内のテキストを自動的に抽出し、クリップボードにコピーするスキルです。

## 対応画像形式

- PNG (.png)
- JPEG (.jpg, .jpeg)
- その他、Claudeが読み取れる画像形式

## 使用例

### 基本的な使い方

```
ユーザー: /path/to/screenshot.png を文字起こしして
```

### プロセス

1. **画像読み取り**: Readツールで画像ファイルを読み込み
2. **テキスト抽出**: Claudeの画像認識機能でテキストを抽出
3. **クリップボードコピー**: `pbcopy`でクリップボードにコピー
4. **結果表示**: 抽出されたテキストを表示

## 注意事項

- 画像の品質や解像度により、抽出精度が変わる可能性があります
- 手書き文字よりも印刷された文字の方が認識精度が高くなります
- 日本語、英語など多言語に対応しています
