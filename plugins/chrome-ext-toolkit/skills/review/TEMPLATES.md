# レビュー返信テンプレート集

Chrome Web Storeレビューへの返信テンプレートです。

> **Note**: 以下のテンプレートは日本語と英語の例を示しています。対応する言語に翻訳して使用してください。`[サポート連絡先]` はプロジェクトのサポート手段（GitHub Issues、メール、サポートページ等）に置き換えてください。

---

## ポジティブレビュー（5つ星）

### 日本語テンプレート

**シンプルな感謝**
```
レビューありがとうございます！気に入っていただけて嬉しいです。今後もより良い拡張機能を目指して改善を続けていきます。
```

**機能への言及あり**
```
[機能名]をお使いいただきありがとうございます！[具体的な使い方]に役立っているとのこと、とても嬉しく思います。今後もご要望があればお知らせください。
```

**フィードバック付きの場合**
```
素晴らしいフィードバックをありがとうございます！ご提案いただいた[機能/改善点]については、今後のアップデートで検討させていただきます。
```

### 英語テンプレート

**Simple thanks**
```
Thank you for your review! We're glad you're enjoying the extension. We'll continue to improve it based on user feedback.
```

**With feature mention**
```
Thank you for using [feature name]! We're happy to hear it's helping with [specific use case]. Let us know if you have any suggestions!
```

**With feedback**
```
Thank you for your wonderful feedback! We'll consider your suggestion about [feature/improvement] in future updates.
```

```

**機能しない**
```

**誤解による低評価**
```

```

```

```

1. Remove and reinstall the extension
2. Update Chrome to the latest version

```

**Misunderstanding**
```
Thank you for your review. Regarding [feature], it works by [correct explanation]. If you have any questions, please feel free to contact us via [support contact], and we'll be happy to help.

## カスタマイズのポイント

返信を作成する際は以下を調整してください：

1. **拡張機能名**を適切に置換
2. **具体的な問題/機能**をレビュー内容に合わせる
3. **サポート連絡先**をプロジェクトのサポート手段に置換（GitHub Issues、メール、サポートページ等）
4. **トーン**をレビューの感情に合わせる
5. **言語**をレビューの言語に合わせる

### トーンの調整

| レビュータイプ | トーン |
|----------------|--------|
| 熱心なファン | 温かく親しみやすい |
| 中立的 | プロフェッショナル |
| イライラしている | 落ち着いて丁寧に |
| 怒っている | 誠実で謙虚に |
