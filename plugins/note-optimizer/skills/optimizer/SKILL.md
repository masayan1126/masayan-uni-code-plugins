# note記事最適化スキル

noteに最適化された記事を生成・最適化するスキルです。

FORMS.mdのフォーム情報を使ってユーザーから必要な情報を収集し、REFERENCE.mdのガイドラインに従って記事を最適化します。

## 使い方

1. **既存記事を最適化する場合**: ファイルパスを指定するか、記事内容を直接貼り付けてください
2. **新規記事を作成する場合**: テーマ、メインメッセージ、ターゲット読者を教えてください

特定のディレクトリ構造は不要です。任意のプロジェクトで使用できます。

## 主な機能

- 既存記事のnote最適化
- 新規記事の作成
- 長文記事の分割提案とシリーズ化
- タイトル最適化（25文字以内）
- リード文作成（50文字程度）
- モバイル最適化（改行・段落調整）
- ハッシュタグ自動生成（5〜10個）
