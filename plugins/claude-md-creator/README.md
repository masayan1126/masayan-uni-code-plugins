# claude-md-creator

Create optimal CLAUDE.md files for your projects following best practices.

## Features

- Creates CLAUDE.md files under 60 lines
- Uses WHY/WHAT/HOW structure
- Leverages external imports for detailed documentation
- Focuses on universal, always-applicable content

## Installation

```bash
/plugin marketplace add masayan/masayan-uni-code-plugins
/plugin install claude-md-creator@masayan-uni-plugins
```

## Usage

Invoke the skill in Claude Code:

```
/claude-md-creator
```

The skill will:
1. Analyze your project structure
2. Identify key components
3. Create a minimal, focused CLAUDE.md

## Skill Files

- `SKILL.md` - Main skill definition
- `PRINCIPLES.md` - WHY/WHAT/HOW design principles
- `TEMPLATES.md` - CLAUDE.md template structure
- `WORKFLOW.md` - Step-by-step creation workflow
