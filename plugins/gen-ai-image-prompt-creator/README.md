# Gen Image Prompt Creator Plugin

Generate image prompts for various use cases with Nano Banana Pro.

## Included Skills

| Skill | Description | Keywords |
|-------|-------------|----------|
| character | Original character designs | キャラクター作成, キャラデザ, オリキャラ |
| engineer-manga | 4-panel engineer humor comics | 漫画作成, エンジニア漫画, 4コマ, あるある |
| pixta-metadata | PIXTA listing metadata | PIXTAタグ, PIXTA出品, ストックイラスト登録 |
| profile-sns | Profile icons and SNS images | プロフィール画像, アイコン作成, SNS用画像 |
| tech-diagram | Technical explanation infographics | 図解作成, テック図解, AI解説図 |
| thumbnail | Blog and YouTube thumbnails | サムネイル作成, サムネ, YouTube用画像 |

## Output Directories

Generated prompts and images are saved to `outputs/` subdirectories.
