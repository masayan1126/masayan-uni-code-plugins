## Nano Banana Pro 用プロンプト

```
【画風】手書き・ミニマル調
- 手描き風ドゥードルスタイル
- 黒インクの輪郭線
- 純白のフラット塗りのみ
- 複雑なテクスチャや影は一切なし
- 物理的なステッカーのような太い白縁取り
- 極限までシンプルでクリーンなデザイン

【キャラクター/モチーフ】シロハラインコ、バナナ、MacBook

【外見設定】
- シロハラインコ: 丸っこいフォルム、白いお腹、緑の羽、オレンジの頭
- バナナ: 大きめのバナナ、シンプルな曲線
- MacBook: 開いた状態、リンゴマーク
- 線のタッチ: 黒インク、落書き風
- 塗り: 純白のみ、グラデーションなし

【構図】
- シロハラインコがバナナの上にちょこんと乗っている
- MacBookの画面を覗き込んでいる姿
- バナナが椅子のような役割

【ポーズ】前のめりで画面を見つめる、興味津々な姿勢
【表情】目をキラキラさせて集中、くちばしは少し開いている

【レイアウト・構図】
- アスペクト比: 16:9
- 背景: ソリッドなティール（青緑）単色、グラデーションなし
- 主要オブジェクトの周りに太い白のステッカー風ボーダー
- テキスト、文字、副次的オブジェクトは配置しない

【補足指示】
- ステッカーとして切り抜けるようなデザイン
- シロハラインコの特徴（白い胸、緑の羽）をシンプルに表現
- ユーモラスで親しみやすい雰囲気
- プロフェッショナルかつクリエイティブな仕上がり
```
