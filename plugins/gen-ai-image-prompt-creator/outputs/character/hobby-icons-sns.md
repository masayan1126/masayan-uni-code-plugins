# 趣味アイコン SNS用 - 手書き・ミニマル調

## プロンプト（日本語）

```
【画風】手書き・ミニマル調
- 手描き風ドゥードルスタイル
- 黒インクの輪郭線
- 純白のフラット塗りのみ
- 複雑なテクスチャや影は一切なし
- 物理的なステッカーのような太い白縁取り
- 極限までシンプルでクリーンなデザイン

【キャラクター/モチーフ】趣味を表すアイコン群
- シロハラインコ（白い胸、緑の羽をシンプルに表現）
- コザクラインコ（ピーチフェイス、緑の体をシンプルに）
- オカメインコ（オレンジのチーク、冠羽をシンプルに）
- 総合格闘技オープンフィンガーグローブ
- MacBook（リンゴマーク付き）
- コーヒーカップ（湯気付き）
- ポケモンカード（モンスターボール模様）
- カラオケマイク
- 漫画（開いた本、吹き出し付き）

【外見設定】
- 形状: 各アイテムをアイコン的にシンプル化
- 特徴: 最小限の線と形で各アイテムを識別可能に
- 線のタッチ: 黒インク、落書き風
- 塗り: 純白のみ、グラデーションなし

【レイアウト・構図】
- アスペクト比: 1:1（SNSアイコン用正方形）
- 背景: ソリッドなティール（青緑）単色、グラデーションなし
- 中央に人物を配置するスペースを空ける
- 周囲にインコ3羽と趣味アイテムを散りばめる
- 各オブジェクトの周りに太い白のステッカー風ボーダー
- テキスト、文字は配置しない

【補足指示】
- 人物は後から別途貼り付け予定のため、中央にスペースを確保
- ステッカーとして切り抜けるようなデザイン
- 複雑な要素は排除し、本質的な形のみで表現
- プロフェッショナルかつクリエイティブな仕上がり
- 各アイテムがバランスよく配置される
```
