# シロハラインコ - クレヨン/色鉛筆風イラスト

## 用途
イラスト作品（観賞用、額装向け）

## プロンプト

```
【画風】スケッチ調 - クレヨン/色鉛筆風
- 温かみのある手描き感
- ざらついた輪郭、不均一で味のある線
- 重ね塗り、ストローク跡が残る
- 鮮やかだが柔らかい発色
- 画用紙の質感

【モチーフ】シロハラインコ（White-bellied Caique）

【インコの特徴】
- 種類: シロハラインコ（White-bellied Caique）
- 体色: オレンジ〜アプリコット色の頭、純白の胸（最大の識別ポイント）、鮮やかな緑の翼と背中、ピンクのくちばしと足
- 識別ポイント: 白い胸とオレンジ頭のコントラスト、ずんぐり丸い体型
- 表現ポイント: オレンジと緑を重ね塗りで鮮やかに、白い胸は紙の色を活かしつつ柔らかいタッチで

【ポーズ】ホッピング（跳ねるポーズ）
- 両足が地面から浮いている
- 翼を少し広げてバランスを取る
- 活発で元気いっぱいな動きの瞬間を捉える

【表情】首をかしげてかわいい
- 好奇心に満ちた目
- 少し口を開けて楽しそう
- 愛嬌のある仕草

【構図】
- アスペクト比: 3:4（縦長、イラスト向け）
- 背景: 淡いクリーム色の画用紙、余白を活かしたシンプルな背景
- 視点: 正面やや斜め、インコの動きを捉える角度

【スケッチ技法】
- 線: 色鉛筆の輪郭線、オレンジと緑で、やや太めでラフ、手描きの温かみ
- 塗り: 重ね塗り、ストローク跡を残す、はみ出し許容、羽毛の質感を表現
- 質感: 画用紙のざらつき、色鉛筆の木目感、クレヨンのワックス感

【補足指示】
- 絵本のような温かみと親しみやすさ
- シロハラインコの元気で活発な性格を表現
- 子供にも大人にも愛されるデザイン
- 額装して飾れるようなアート作品としての完成度
```
