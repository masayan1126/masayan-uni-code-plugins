## 設計メモ

### 図解タイプの選定理由
- 「比較表型」を採用：Web時代とAI時代の対比を視覚的に表現
- 左右に並べることで「同じ構造だ」という洞察を強調

### コンセプト
- O'Reillyの洞察：MCPとスキルは初期Webの「ソースを表示」と同じ役割
- 「参加のアーキテクチャ」の具体例として提示
- 技術者でなくても「なるほど」と思える対比構造

### MCP/スキルの定義（参考）
- **MCP**: AIシステムに外部サービス（DB、API、GitHub等）へのアクセスを提供する小規模サーバー
- **スキル**: Claudeに特定の操作方法を教える、平易な言語による指示書の集合体
