# 画像生成AIプロンプトテクニック6選

> 作成日: 2024-12-29
> テイスト: 手書き風

## 図解プロンプト

```
【タイトル】画像生成AIプロンプトテクニック6選（画像上部に大きく配置）

【図解タイプ】概念図（階層構造）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】マーカーペンの色味（青、オレンジ、緑、赤）、白または薄いクリーム色の背景
【レイアウト】上下2段構成、上段に必須要素3つ、下段にオプション要素3つ

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、図解を指し示すポーズ

【構成要素】
- 上段（必須レベル）: 3つの要素を横並び、オレンジ/赤のマーカーで強調
  - ①優先順位
  - ②具体性
  - ③光源
- 下段（オプション）: 3つの要素を横並び、青/緑のマーカー
  - ④視点
  - ⑤画風
  - ⑥色調・色彩
- 上段と下段の間に手描きの波線と「★必須！」「あると便利」のラベル

【テキスト内容】
上段ラベル: 「★必須！」（手書き文字風）
①優先順位: 「主題を冒頭に」
②具体性: 「詳細に記述」
③光源: 「AI感を抑える鍵」

下段ラベル: 「あると便利」（手書き文字風）
④視点: 「俯瞰・ズーム」
⑤画風: 「印象派・浮世絵」
⑥色調: 「アースカラー」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードやノートに描いたような質感
- マーカーペンやサインペンのタッチ
- 線は少しブレがあり、完璧でない有機的な形
- 矢印や囲み枠も手描き風
- 日本語テキストも手書きフォント風
- 余白を十分に確保
- 親しみやすくカジュアルな雰囲気
- キャラクターが解説している構図
```
