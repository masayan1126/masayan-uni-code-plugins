# 画像生成プロンプト作成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】画像生成プロンプト作成（画像上部に大きく配置）

【図解タイプ】概念図（対話フロー）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】紫・青・オレンジのマーカー、白またはクリーム色の背景
【レイアウト】対話形式の4ステップ質問と最終プロンプト

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、対話フローを指し示しながら解説している構図

【構成要素】
- 左側: 4つの対話ステップ（質問形式）
- 右側: 生成されるプロンプト例
- 下部: プロンプト構成要素

【テキスト内容】
左側「対話フロー」:
  吹き出し形式で4つの質問:

  Q1: 「被写体は何ですか？」
  選択肢: キャラクター / 風景 / プロダクト

  Q2: 「どんな雰囲気ですか？」
  選択肢: モダン / 温かみ / クール

  Q3: 「色調の好みは？」
  選択肢: ダーク / ライト / ビビッド

  Q4: 「使用ツールは？」
  選択肢: Midjourney / DALL-E / SD

右側（緑）「生成プロンプト」:
  コードブロック風:
  ```
  modern tech workspace,
  floating code snippets,
  holographic UI elements,
  dark background,
  cyan and purple accent,
  isometric view,
  4K, highly detailed
  --ar 16:9 --v 6
  ```

下部「プロンプト構成要素」:
  5つの要素を横に並べる:
  - 「被写体」
  - 「スタイル」
  - 「色調」
  - 「構図」
  - 「品質」

下部のメッセージ:
  「対話形式で最適なプロンプトを生成」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 対話の吹き出しで質問形式を表現
- 最終プロンプトはコードブロック風
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「概念図型（対話フロー）」を採用：対話形式でプロンプト生成
- 質問→回答→プロンプトの流れを可視化

### コンセプト
- 対話形式で被写体・スタイル・色調・構図を選択
- ツール別に最適化されたプロンプト生成
- 語彙辞書でキーワードを補完
