## 設計メモ

### 図解タイプの選定理由
- 「一覧表型」を採用：22ユースケースを網羅的に見渡せる
- 3列構成で各カテゴリを並列表示
- 16:9横長で情報量を確保

### コンセプト
- 全ユースケースを一目で把握できる俯瞰図
- 3カテゴリの色分けで構造を明確化
- 「定型作業→自動化→創造的時間」のメッセージ

### レイアウト詳細
- 左列（青）: 開発系10個 - 最も項目が多いが技術者向けで馴染みやすい
- 中央列（オレンジ）: コンテンツ系10個 - クリエイター向け
- 右列（緑）: ドキュメント系2個 - 項目少ないので下部にキーメッセージ追加
