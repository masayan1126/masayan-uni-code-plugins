# 音楽生成プロンプト作成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】音楽生成プロンプト作成（画像上部に大きく配置）

【図解タイプ】概念図（対話フロー）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】オレンジ・紫・緑のマーカー、白またはクリーム色の背景
【レイアウト】対話形式の4ステップ質問と最終プロンプト

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、対話フローを指し示しながら解説している構図

【構成要素】
- 左側: 4つの対話ステップ（質問形式）
- 右側: 生成されるプロンプト例
- 下部: 用途別テンプレート

【テキスト内容】
左側「対話フロー」:
  吹き出し形式で4つの質問:

  Q1: 「どんなジャンルですか？」
  選択肢: エレクトロニック / ローファイ / ポップ

  Q2: 「雰囲気は？」
  選択肢: リラックス / 集中 / 楽しい

  Q3: 「テンポの好みは？」
  選択肢: スロー / ミディアム / アップテンポ

  Q4: 「ボーカルは？」
  選択肢: あり / なし（BGM）

右側（緑）「生成プロンプト」:
  コードブロック風:
  ```
  lofi hip hop,
  study beats,
  chill instrumental,
  mellow piano,
  warm vinyl crackle,
  soft drums,
  relaxing atmosphere,
  85 BPM,
  no vocals
  ```

下部「用途別テンプレート」:
  4つのアイコンを横に並べる:
  - 「BGM」（ヘッドホン）
  - 「ジングル」（ベル）
  - 「ポップソング」（マイク）
  - 「シネマティック」（映画カチンコ）

下部のメッセージ:
  「Suno等向けに最適化されたプロンプト」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 音符アイコンを散りばめて音楽感を演出
- 対話の吹き出しで質問形式を表現
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「概念図型（対話フロー）」を採用：画像プロンプトと同様の対話形式
- 音楽特有の要素（ジャンル、テンポ、BPM）を含む

### コンセプト
- 対話形式でジャンル・ムード・テンポ・楽器を選択
- Suno、Udioなどの音楽生成AI向け
- BGM制作、ジングル、ポップソングなど用途別テンプレート
