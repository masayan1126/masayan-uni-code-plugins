# FAQ・想定質問生成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】FAQ・想定質問生成（画像上部に大きく配置）

【図解タイプ】概念図（変換プロセス）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】青・オレンジ・緑のマーカー、白またはクリーム色の背景
【レイアウト】コンテンツからFAQへの変換を中央に、4つの抽出ソースを左に配置

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、FAQ生成を指し示しながら解説している構図

【構成要素】
- 左側: 4つの抽出ソース（コンテンツ、プロダクト、フィードバック、反論）
- 中央: Agent Skill（質問抽出・分類）
- 右側: 構造化されたFAQ

【テキスト内容】
左側（青）「抽出ソース」:
  4つの入力:
  1. 「ブログ記事」
  2. 「プロダクト情報」
  3. 「問い合わせ履歴」
  4. 「想定される反論」

中央:
  「Agent Skill」
  - 質問抽出
  - カテゴリ分類
  - 優先度付け
  - 回答生成

  矢印「→」

右側（緑）「FAQ」:
  構造化されたFAQ:

  「よくある質問」

  Q1: 「〇〇とは何ですか？」
  A1: 「〇〇は...」

  Q2: 「料金はいくらですか？」
  A2: 「料金プランは...」

  Q3: 「セキュリティは大丈夫？」
  A3: 「暗号化して...」

下部のメッセージ:
  「ユーザーの疑問を先回りして解消」
  「問い合わせ削減・CV率向上」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 4つの抽出ソースを左に
- Q&A形式のFAQを右に
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「概念図型」を採用：複数ソースからFAQへの変換プロセス
- 抽出ソースの多様性を強調

### コンセプト
- コンテンツから想定質問を自動抽出
- 反論・懸念への対応も含む
- 「先回りして解消」がキーメッセージ
