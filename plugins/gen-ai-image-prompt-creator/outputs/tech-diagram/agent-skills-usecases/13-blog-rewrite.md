# ブログ記事リライト - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】ブログ記事リライト（画像上部に大きく配置）

【図解タイプ】フローチャート（循環型）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】赤（課題）・青（分析）・緑（改善）のマーカー、白またはクリーム色の背景
【レイアウト】4ステップの循環フロー

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、リライトサイクルを指し示しながら解説している構図

【構成要素】
- ステップ1: 記事分析（虫眼鏡アイコン）
- ステップ2: 課題特定（×アイコン）
- ステップ3: 改善施策（工具アイコン）
- ステップ4: 効果測定（グラフアイコン）
- 中央: 「検索順位UP」

【テキスト内容】
ステップ1（青）「記事分析」:
  - SEO観点分析
  - コンテンツ品質
  - 競合記事比較

ステップ2（赤）「課題特定」:
  - 検索順位低下
  - 情報の陳腐化
  - クリック率低下
  - 読了率・滞在時間

ステップ3（緑）「改善施策」:
  - タイトル改善
  - 見出し改善
  - 本文更新・拡充
  - 内部リンク最適化

ステップ4（青）「効果測定」:
  - KPI設定
  - 順位追跡
  - PV・CTR確認

中央:
  「検索順位UP」
  上向き矢印

下部のメッセージ:
  「古い記事を最新化して検索順位を改善」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 4ステップを循環型で配置
- 中央にゴール（検索順位UP）
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「フローチャート型（循環）」を採用：リライトは継続的なサイクル
- 効果測定→分析に戻る循環構造

### コンセプト
- 既存記事の価値を最大化
- データに基づく改善サイクル
- 「検索順位UP」がゴール
