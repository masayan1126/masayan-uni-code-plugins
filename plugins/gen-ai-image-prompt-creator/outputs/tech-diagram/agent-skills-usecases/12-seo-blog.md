# SEOに最適化されたブログ記事作成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】SEOに最適化されたブログ記事作成（画像上部に大きく配置）

【図解タイプ】フローチャート（左から右への流れ）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】青・オレンジ・緑のマーカー、白またはクリーム色の背景
【レイアウト】左から右への5ステップフロー

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、記事作成フローを指し示しながら解説している構図

【構成要素】
- ステップ1: メモ・ネタ（ノートアイコン）
- ステップ2: 構成設計（見出しアイコン）
- ステップ3: 本文作成（ペンアイコン）
- ステップ4: SEO最適化（検索アイコン）
- ステップ5: 公開（ロケットアイコン）
- 下部: 各ステップの詳細

【テキスト内容】
ステップ1（青）「メモ・ネタ」:
  - 雑多な情報
  - アイデアメモ
  - 参考資料

ステップ2（青）「構成設計」:
  - H1〜H3見出し設計
  - 記事テンプレート選択
  - アウトライン作成

ステップ3（オレンジ）「本文作成」:
  - リード文
  - 本文（です・ます調/だ・である調）
  - まとめ

ステップ4（緑）「SEO最適化」:
  - タイトル最適化
  - メタディスクリプション
  - キーワード配置
  - 内部リンク

ステップ5（緑）「公開」:
  - 公開前チェック
  - 画像・図解配置
  - 多言語対応

下部のメッセージ:
  「メモから高品質な記事を自動生成」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 5ステップを左から右へ
- 各ステップにシンプルなアイコン
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「フローチャート型」を採用：記事作成の一連の流れを可視化
- 5ステップで全体像を把握

### コンセプト
- メモから記事への変換プロセス
- SEO最適化を組み込んだワークフロー
- 多言語対応も含む
