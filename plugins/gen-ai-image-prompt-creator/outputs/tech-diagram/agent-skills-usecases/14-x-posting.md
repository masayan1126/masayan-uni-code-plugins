# X投稿最適化 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】X投稿最適化（画像上部に大きく配置）

【図解タイプ】概念図（4要素の構成）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】青（Xカラー）・オレンジ・緑のマーカー、白またはクリーム色の背景
【レイアウト】中央に投稿、周囲に4つの最適化要素を配置

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、投稿最適化のポイントを指し示しながら解説している構図

【構成要素】
- 中央: X投稿（スマホ/ツイートアイコン）
- 4つの最適化要素:
  1. フック（冒頭文）
  2. フォーマット
  3. ハッシュタグ
  4. 投稿タイミング
- 下部: エンゲージメント指標

【テキスト内容】
中央:
  「X投稿」
  スマホアイコン with ツイート

要素1（青）「フック」:
  - 冒頭で惹きつける
  - 「〇〇を図解してみた」
  - 「5分でわかる〇〇」

要素2（オレンジ）「フォーマット」:
  - スレッド投稿
  - 引用RT
  - 画像・動画付き

要素3（緑）「ハッシュタグ」:
  - 適切なタグ選定
  - トレンド活用
  - ニッチタグも

要素4（紫）「タイミング」:
  - 最適な投稿時間
  - フォロワーのアクティブ時間
  - 自動投稿

下部（エンゲージメント）:
  「いいね」「RT」「リプライ」「インプレッション」
  上向き矢印

下部のメッセージ:
  「エンゲージメント率を向上させる」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 中央から4方向に放射
- X（旧Twitter）のブランドカラーを意識
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「概念図型」を採用：投稿を中心に4つの最適化要素を放射状に配置
- エンゲージメント向上がゴール

### コンセプト
- フック、フォーマット、ハッシュタグ、タイミングの4要素
- アルゴリズムを意識した投稿最適化
- 自動投稿との連携も視野に
