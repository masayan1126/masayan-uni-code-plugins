# 会議のアジェンダ作成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】会議のアジェンダ作成（画像上部に大きく配置）

【図解タイプ】フローチャート（左から右への流れ）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】青・オレンジ・緑のマーカー、白またはクリーム色の背景
【レイアウト】会議前→会議中→会議後の3フェーズ

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、アジェンダ作成フローを指し示しながら解説している構図

【構成要素】
- フェーズ1: 会議前（アジェンダ設計）
- フェーズ2: 会議中（ファシリテーション）
- フェーズ3: 会議後（フォローアップ）
- 下部: 会議タイプ別テンプレート

【テキスト内容】
フェーズ1（青）「会議前」:
  アイコン: カレンダー
  - 目的・ゴール設定
  - 論点整理
  - 時間配分（タイムボックス）
  - 参加者・役割

フェーズ2（オレンジ）「会議中」:
  アイコン: 人々
  - 進行管理
  - 決定事項の明確化
  - アクションアイテム抽出

フェーズ3（緑）「会議後」:
  アイコン: チェックリスト
  - 議事録作成
  - 共有・公開
  - 進捗追跡

下部「会議タイプ別テンプレート」:
  5つを横に並べる:
  - 「デイリースタンドアップ」
  - 「スプリントレビュー」
  - 「1on1」
  - 「キックオフ」
  - 「ブレスト」

下部のメッセージ:
  「ゴールの決まっていない会議はやらない」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- 3フェーズを左から右へ
- 各フェーズを色分け
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「フローチャート型」を採用：会議前→中→後の3フェーズ
- 会議タイプ別テンプレートを併記

### コンセプト
- 「ゴールの決まっていない会議はやらない」がキーメッセージ
- 目的・論点・時間配分の事前設計が重要
- 透明性のある情報共有
