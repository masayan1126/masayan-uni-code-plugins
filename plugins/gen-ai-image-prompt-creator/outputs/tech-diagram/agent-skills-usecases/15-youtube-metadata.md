# YouTubeメタデータ生成 - 図解プロンプト

> **重要**: このプロンプトを画像AIに入力する際は、キャラクターの元画像を一緒に添付してください。

## プロンプト

```
【タイトル】YouTubeメタデータ生成（画像上部に大きく配置）

【図解タイプ】フローチャート（左から右への流れ）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】赤（YouTube）・青・緑のマーカー、白またはクリーム色の背景
【レイアウト】動画から4つのメタデータ要素を生成するフロー

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、メタデータ生成フローを指し示しながら解説している構図

【構成要素】
- 左側: 動画コンテンツ（再生アイコン）
- 中央: Agent Skill（変換）
- 右側: 4つのメタデータ要素を縦に並べる
- 下部: 最適化のポイント

【テキスト内容】
左側（赤）「動画コンテンツ」:
  再生アイコン
  - 動画内容
  - スクリプト
  - キーワード

中央:
  「Agent Skill」
  大きな矢印「→」
  「自動生成」

右側（4つの要素を縦に）:
  1.「タイトル」（赤で強調）
    - SEO最適化
    - CTR向上
    - A/Bテスト案

  2.「説明文」（青）
    - 最初の3行が重要
    - タイムスタンプ
    - リンク・CTA

  3.「タグ・ハッシュタグ」（緑）
    - キーワードリサーチ
    - 競合分析
    - 適切な数

  4.「サムネイルテキスト」（オレンジ）
    - インパクトのある文言
    - 視認性重視

下部のメッセージ:
  「検索流入とクリック率を向上」

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードに描いたような質感
- マーカーペンのタッチ（線に少しブレがある）
- 親しみやすい雰囲気
- 日本語テキスト
- YouTubeの赤を意識
- 4つの要素を縦に並べて見やすく
- 余白を十分に確保
- 男の子とインコが一緒に解説している構図
```

## 設計メモ

### 図解タイプの選定理由
- 「フローチャート型」を採用：動画→メタデータ生成の流れ
- 4つのメタデータ要素を縦に並べて網羅

### コンセプト
- YouTube SEOを意識したメタデータ生成
- タイトル、説明文、タグ、サムネイルテキストの4要素
- 「検索流入とクリック率向上」がゴール
