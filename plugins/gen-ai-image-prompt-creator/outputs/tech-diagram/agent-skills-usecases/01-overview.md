## プロンプト

```
【タイトル】Agent Skillsユースケース全体像（画像上部に大きく配置）

【図解タイプ】概念図（中心から放射状に3カテゴリが展開）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
```

## 設計メモ

### 図解タイプの選定理由
- 「概念図型」を採用：全体像を俯瞰し、3カテゴリの関係性を視覚化
- 中心から放射状のレイアウトで、核心メッセージを強調

### コンセプト
- Agent Skillsの価値提案：「定型作業の自動化で創造的時間を確保」
- 22ユースケースを3カテゴリに整理して一目で把握できるように
- 各カテゴリの個数（10/10/2）を明示してバランスを示す

### カラースキーム
- **青（開発系）**: 技術・コードをイメージ
- **オレンジ（コンテンツ系）**: 創造性・表現をイメージ
- **緑（ドキュメント系）**: 知識・整理をイメージ

### シリーズとしての位置づけ
- この図解がシリーズ全体の目次・俯瞰図として機能
- 以降の個別ユースケース図解への導線となる
