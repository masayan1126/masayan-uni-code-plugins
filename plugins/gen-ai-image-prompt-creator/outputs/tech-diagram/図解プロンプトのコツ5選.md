# 図解プロンプトのコツ5選

> 作成日: 2024-12-29
> テイスト: 手書き風

## 図解プロンプト

```
【タイトル】図解プロンプトのコツ5選（画像上部に大きく配置）

【図解タイプ】概念図（階層構造）
【テイスト】手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】マーカーペンの色味（オレンジ、青、緑）、白または薄いクリーム色の背景
【レイアウト】上下2段構成、上段に必須要素3つ、下段にオプション要素2つ

【解説キャラクター】添付画像のキャラクター（男の子とインコたち）を右下に配置、ホワイトボードを指し示すポーズ

【構成要素】
- 上段（必須レベル）: 3つの要素を横並び、オレンジのマーカーで強調
  - ①優先順位
  - ②具体性
  - ③レイアウト
- 下段（オプション）: 2つの要素を横並び、青/緑のマーカー
  - ④テイスト
  - ⑤カラースキーム
- 上段と下段の間に手描きの波線と「★必須！」「あると良い」のラベル

【テキスト内容】
上段ラベル: 「★必須！」（手書き文字風、赤マーカー）

①優先順位:
- 見出し: 「タイトル・主題を冒頭に」
- 補足: 「最初に書いた要素がAIに強く影響」
- アイコン: 上向き矢印と「1st」

②具体性:
- 見出し: 「要素を詳細に記述」
- 補足: 「曖昧な指示→曖昧な出力」
- アイコン: 虫眼鏡

③レイアウト:
- 見出し: 「情報の流れを明示」
- 補足: 「左→右、上→下、放射状」
- アイコン: 矢印のフロー図

下段ラベル: 「あると良い」（手書き文字風、青マーカー）

④テイスト:
- 見出し: 「クリーン or 手書き風」
- 補足: 「雰囲気を左右する」
- アイコン: ペン/ブラシ

⑤カラースキーム:
- 見出し: 「配色パターン指定」
- 補足: 「テック系、パステル調など」
- アイコン: カラーパレット

【スタイル指示】
- 手書き風スケッチ、ラフな走り書き
- ホワイトボードやノートに描いたような質感
- マーカーペンやサインペンのタッチ
- 線は少しブレがあり、完璧でない有機的な形
- 矢印や囲み枠も手描き風
- 日本語テキストも手書きフォント風
- 余白を十分に確保
- 親しみやすくカジュアルな雰囲気
- キャラクター（男の子とインコ）が解説している構図
```
