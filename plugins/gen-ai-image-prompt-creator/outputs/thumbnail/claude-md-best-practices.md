## Nano Banana Pro 用プロンプト

> **注意**: 画像生成時に、シロハラインコのベースイラストを添付してください。

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】「CLAUDE.md ベストプラクティス」「60行以下で性能最大化」
【メインビジュアル】添付画像のシロハラインコキャラクターとmacbook
【背景】白（単色、シンプル）
【配色】白背景、黒テキスト、キャラクターの色はそのまま活かす
【スタイル】人物・キャラ型

【キャラクター指示】
- 添付画像のキャラクターをそのまま使用
- 画風・タッチ・色を維持
- ポーズは元画像そのままではなく、以下からランダムに選んで変更すること
  - 寝る、走る、頑張る、飛ぶ、びっくりする、喜ぶ、考える、手を振る など

【レイアウト】
- 下部中央にシロハラインコキャラクター
  - キャラクターはサムネの中央（左右の軸）に配置すること
- 上部から中部中央にタイトルテキスト
- 装飾は最小限、クリーンでシンプルに

【テキスト配置】
- メインタイトル「CLAUDE.md ベストプラクティス」: 上部中央、太いゴシック体、黒色、大きめ
- メインタイトルにアンダーラインを引いておしゃれに
- サブタイトル「60行以下で性能最大化」: メインの下、やや小さめ、黒色
- サブタイトルとメインの間にほんの少しだけ余白を入れること
- シンプルで読みやすいフォント

【補足指示】
- クリック率を意識したデザイン
- 添付キャラクターの画風・色を忠実に再現
- シンプルでクリーンなデザイン
- 日本語テキスト
- 余計な装飾は入れない
- テキストやキャラ画像の配置、整列、文字サイズ、バランスに気を使い、均等のとれた配置にすること
```
