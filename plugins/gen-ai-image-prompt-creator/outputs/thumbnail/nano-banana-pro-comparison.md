# Nano Banana Pro 比較記事サムネイル

## プロンプト

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】なし（キャラクター・オブジェクトのみ）
【メインビジュアル】シロハラインコ、MacBook、バナナの組み合わせ
【背景】明るいクリーム色からライトイエローへのグラデーション
【配色】黄色（バナナ）、シルバー（MacBook）、白・グレー・緑（シロハラインコ）、温かみのある色調
【スタイル】オブジェクト中心型

【レイアウト】
- 中央にMacBookを配置（画面が見える角度）
- MacBookの上または横にシロハラインコが乗っている
- バナナを複数本、MacBookの周りに散りばめる
- 全体的に明るく親しみやすい雰囲気

【構図詳細】
- シロハラインコ: 可愛らしいポーズ、好奇心旺盛な表情、MacBookの画面を覗き込んでいる
- MacBook: シルバーのMacBook Pro、画面には柔らかい光
- バナナ: 黄色く熟したバナナ、2〜3本を自然に配置

【補足指示】
- テキストは入れない、画像のみで構成
- 明るく楽しい雰囲気
- クリーンでモダンなデザイン
- シロハラインコの羽の質感をリアルに
- 全体的にポップで目を引くビジュアル
- コンテスト応募用として完成度の高い仕上がり
```

## 配色詳細

| 要素 | カラー |
|------|--------|
| 背景 | #FFF8E7 → #FFFACD (クリーム〜ライトイエロー) |
| バナナ | #FFE135 (鮮やかな黄色) |
| MacBook | #C0C0C0 (シルバー) |
| シロハラインコ | 白、グレー、薄緑のグラデーション |

## デザインコンセプト

Nano Banana Proの「バナナ」をモチーフに、AIツールを使う様子をシロハラインコとMacBookで表現。テキストなしでも一目で「バナナ」「テック」「可愛い」が伝わるビジュアル。コンテスト応募用として、クオリティと独自性を重視したデザイン。
