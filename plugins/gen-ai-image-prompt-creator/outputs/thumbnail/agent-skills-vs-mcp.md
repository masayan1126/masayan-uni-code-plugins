# サムネイルプロンプト: Agent Skills vs MCP

## 記事情報
- **タイトル**: 【Claude Code】Agent SkillsとMCPの違いを徹底解説 - 補完関係を理解して使いこなす
- **プラットフォーム**: ブログ
- **スタイル**: 比較型（補完関係バージョン）

---

## プロンプト

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】「Skills vs MCP」「補完関係を理解する」
【メインビジュアル】左右対比のレイアウト、中央でパズルピースのように噛み合う2つの要素
【背景】左側オレンジ系（Skills：Anthropicカラー）、右側緑系（MCP：接続・通信のイメージ）、中央でグラデーション融合
【配色】オレンジ（左）と緑（右）で対比、中央は両色が混ざり合う
【スタイル】比較型（補完関係）

【レイアウト】
- 左側: 「Skills」のテキストとドキュメント・コマンドのアイコン
- 右側: 「MCP」のテキストとプラグ・接続のアイコン
- 中央: パズルピースまたは歯車が噛み合うビジュアル（対決ではなく協調）
- 上部に「Claude Code」のロゴ風テキスト
- 下部に「徹底解説」のサブテキスト

【テキスト配置】
- 「Claude Code」: 画像上部中央、小さめ、白色
- 「Skills」: 左側中央、太いゴシック体、オレンジ色
- 「MCP」: 右側中央、太いゴシック体、緑色
- 中央: 連結を示すアイコン（双方向矢印やパズル）
- 「補完関係を理解する」: 画像下部、白色、やや小さめ

【補足指示】
- クリック率を意識したデザイン
- 対決ではなく「補完・協調」を表現
- 両者が連携して機能するイメージ
- テック感がありつつ分かりやすいデザイン
- 日本語テキスト（Skills、MCPは英語表記）
- Anthropicのブランドカラー（オレンジ）を意識
```

## デザインコンセプト

「VS」ではなく「&」や「+」のイメージ。Agent SkillsとMCPが対立するのではなく、それぞれの強みを活かして補完し合う関係性を視覚的に表現。パズルのピースが噛み合う、または歯車が連動するようなビジュアルで「一緒に使うことで真価を発揮する」というメッセージを伝える。
