# AIブラウザ活用事例20選 - サムネイルプロンプト

## プロンプト

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】AIブラウザ活用事例20選
【メインビジュアル】添付画像のキャラクター
【背景】白（単色、シンプル）
【配色】白背景、黒テキスト、キャラクターの色はそのまま活かす
【スタイル】人物・キャラ型

【キャラクター指示】
- 添付画像のインコキャラクターをそのまま使用
- 画風・タッチ・色を維持
- ポーズ: パソコンを操作している、または考えているポーズ

【レイアウト】
- 下部中央にインコキャラクター
  - キャラクターはサムネの中央（左右の軸）に配置すること
- 上部から中部中央にタイトルテキスト
- 装飾は最小限、クリーンでシンプルに

【テキスト配置】
- メインタイトル: 上部中央、太いゴシック体、黒色、大きめ
- 「AIブラウザ活用事例20選」
- メインタイトルの下に手書き風のラフなアンダーライン（マーカーで引いたような太めのライン、アクセントカラーで強調）
- サブタイトル: 「すぐ実践できるプロンプトも紹介」メインの下、やや小さめ、黒色
- サブタイトルとメインの間にほんの少しだけ余白を入れること
- シンプルで読みやすいフォント

【補足指示】
- クリック率を意識したデザイン
- 添付キャラクターの画風・色を忠実に再現
- シンプルでクリーンなデザイン
- 日本語テキスト
- 余計な装飾は入れない
- テキストやキャラ画像の配置、整列、文字サイズ、バランスに気を使い、均等のとれた配置にすること
```

## 使用方法

1. Nano Banana Proで上記プロンプトを入力
2. インコのベースイラストを添付
3. 画像を生成
