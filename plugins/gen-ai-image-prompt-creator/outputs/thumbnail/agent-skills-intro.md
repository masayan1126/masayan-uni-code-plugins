# Agent Skills入門 サムネイル

## 概要

- **形式**: ブログサムネイル (1200x630)
- **スタイル**: 人物・キャラ型
- **キャラクター**: シロハラインコ
- **タイトル**: Agent Skillsとは？仕組みと基本概念をわかりやすく解説

---

## Nano Banana Pro 用プロンプト

> **注意**: 画像生成時に、シロハラインコのベースイラストを添付してください。

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】「Agent Skillsとは？」「仕組みと基本概念をわかりやすく解説」
【メインビジュアル】添付画像のシロハラインコキャラクター
【背景】白（単色、シンプル）
【配色】白背景、黒テキスト、キャラクターの色はそのまま活かす
【スタイル】人物・キャラ型

【キャラクター指示】
- 添付画像のキャラクターをそのまま使用
- 画風・タッチ・色を維持
- キャラクターのポーズはランダムでおまかせ

【レイアウト】
- 下部中央にシロハラインコキャラクター
    - キャラクターはサムネの中央（左右の軸）に配置すること
- 上部から中部中央にタイトルテキスト
- 装飾は最小限、クリーンでシンプルに

【テキスト配置】
- メインタイトル「Agent Skillsとは？」: 左上、太いゴシック体、黒色、大きめ
- サブタイトル「仕組みと基本概念をわかりやすく解説」: メインの下、やや小さめ、黒色
- サブタイトルとメインの間にほんの少しだけ余白を入れること
- シンプルで読みやすいフォント

【補足指示】
- クリック率を意識したデザイン
- 添付キャラクターの画風・色を忠実に再現
- シンプルでクリーンなデザイン
- 日本語テキスト
- 余計な装飾は入れない
```
