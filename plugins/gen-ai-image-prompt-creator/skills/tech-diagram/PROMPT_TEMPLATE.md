# 図解プロンプトテンプレート

> **重要**: プロンプトを画像AIに入力する際は、必ずキャラクターの元画像を一緒に添付してください。

## 基本構造

```
【タイトル】{title}（画像上部に大きく配置）

【図解タイプ】{diagram_type}
【テイスト】{taste} ※下記から選択
  - クリーン: クリーンでモダンなインフォグラフィック、整った線、デジタルイラスト風
  - 手書き風: 手書き風スケッチ、ラフな走り書き、ホワイトボードに描いたような、マーカーペンのタッチ、親しみやすい雰囲気
【カラースキーム】{color_scheme}
【レイアウト】{layout}
【説明対象】{topic}

【解説キャラクター】添付画像のキャラクターを解説役として配置。{character_position}

【構成要素】
{elements}

【テキスト内容】（すべて日本語）
{text_content}

【スタイル指示】
- {taste_style} ※選択したテイストに応じた指示
- 日本語テキスト
- 余白を十分に確保
- 添付画像のキャラクターが解説している構図
{style_notes}
```

## 図解タイプ別テンプレート

### フローチャート型

```
【タイトル】{title}（画像上部に大きく配置）

【図解タイプ】フローチャート
【カラースキーム】青・紫グラデーション、ダークテーマ
【レイアウト】左から右への流れ、各ステップをボックスで表現

【解説キャラクター】添付画像のキャラクターを右下または左下に配置、フローを指し示すポーズ

【構成要素】
- ステップ1: {step1_name}
- ステップ2: {step2_name}
- ステップ3: {step3_name}
- ステップ4: {step4_name}
- 各ステップ間は矢印で接続

【テキスト内容】
ステップ1: 「{step1_description}」
ステップ2: 「{step2_description}」
ステップ3: 「{step3_description}」
ステップ4: 「{step4_description}」

【スタイル指示】
- インフォグラフィック風
- 日本語テキスト
- 矢印は明確に、流れが分かりやすく
- 各ボックスにアイコンを配置
- キャラクターが解説している構図
```

## キャラクター配置のコツ

### 配置パターン
- **右下配置**: 図解を指し示す、説明している構図
- **左下配置**: 図解を見上げている、学んでいる構図
- **吹き出し付き**: キャラクターのセリフで要点を補足

### キャラクター画像の添付について
- 画像AIにプロンプトを入力する際、必ずキャラクターの元画像を一緒に添付する
- プロンプト内では「添付画像のキャラクター」として参照する
- 図解の内容を邪魔しないサイズ・配置にする

### キャラクター構成
添付画像には以下のキャラクターが含まれる：
- **男の子**: 解説役
- **インコ（数羽）**: 一緒に登場

プロンプト生成時は、男の子とインコを両方とも図解に登場させること。
