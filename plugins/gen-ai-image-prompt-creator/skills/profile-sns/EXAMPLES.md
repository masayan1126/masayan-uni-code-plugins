# サンプルプロンプト集

## 例1: プロフィールアイコン（イラスト風エンジニア）

### プロンプト

```
【用途】プロフィールアイコン (400x400、正方形)
【メインモチーフ】エンジニア風のキャラクター、上半身
【スタイル】イラスト風
- アニメ調のタッチ
- はっきりした輪郭線
- ビビッドカラー
【カラースキーム】青系、背景は淡いグラデーション
【背景】水色から白へのグラデーション

【構成】
- 正面やや斜めを向いたキャラクター
- 短い黒髪、丸メガネ
- パーカーを着用
- 自信のある笑顔
- 上半身のみ、中央に配置

【補足指示】
- 小さく表示されても顔が認識できるサイズ
- 円形トリミングを考慮
- テック感のある雰囲気
- 親しみやすい表情
```

## 例2: プロフィールアイコン（フラットデザイン）

### プロンプト

```
【用途】プロフィールアイコン (400x400、正方形)
【メインモチーフ】コードを書くラップトップ、抽象化したシンボル
【スタイル】フラットデザイン
- シンプル、モダン
- 輪郭線なし
- 3色のみ使用
【カラースキーム】濃紺、白、オレンジアクセント
【背景】濃紺（#1E3A5F）単色

【構成】
- 中央にラップトップのシンプルなアイコン
- 画面部分にコードを示す「< >」シンボル
- 上部に小さな光のアクセント

【補足指示】
- ミニマルでプロフェッショナルな印象
- 小さくても認識できるシンプルさ
- ビジネス用途に適した雰囲気
```

## 例3: Xヘッダー（テック系）

### プロンプト

```
【用途】Xヘッダー (1500x500、横長)
【メインビジュアル】コードと幾何学模様が混在する抽象的なデザイン
【スタイル】モダン・テック
【カラースキーム】ダークテーマ、青と紫のグラデーション
【背景】濃紺から黒へのグラデーション

【構成】
- 左側1/3は空ける（プロフィール画像用）
- 中央から右にかけてコードの断片が浮かぶ
- 接続線やノードで技術感を演出
- 右端に向かって光のパーティクル

【テキスト】（任意）
「Building the future with code」右上に小さく配置

【補足指示】
- サイバー感のある雰囲気
- プロフィールアイコンとの調和
- 暗すぎず、適度なコントラスト
```

## 例4: Xヘッダー（クリエイター系）

### プロンプト

```
【用途】Xヘッダー (1500x500、横長)
【メインビジュアル】作品ギャラリー風のコラージュ
【スタイル】ポップ・カラフル
【カラースキーム】パステルカラー多色、白背景
【背景】白ベース、パステルカラーの幾何学模様

【構成】
- 左側1/3は空ける
- 中央から右にかけて、様々なアート作品のサムネイルが浮かぶ
- 作品は斜めに配置してダイナミックに
- 余白を活かしたクリーンなデザイン

【補足指示】
- クリエイティブで楽しい雰囲気
- 作品の多様性を表現
- ポートフォリオ感を演出
```

## 例5: X投稿用（自己紹介カード）

### プロンプト

```
【用途】X投稿用画像 (1200x675、16:9)
【目的】自己紹介カード
【メインビジュアル】キャラクターアイコン＋プロフィール情報
【スタイル】フラットデザイン、カード風
【カラースキーム】青、白、グレー
【背景】薄いグレー、カード形式の白いボックス

【構成】
- 左側にキャラクターアイコン（大きめ）
- 右側にテキスト情報
  - 名前
  - 肩書き
  - スキルアイコン（言語ロゴなど）
  - SNSリンクアイコン
- カード風の影をつけて立体感

【テキスト】
名前: 「山田太郎」
肩書き: 「フルスタックエンジニア」
スキル: Python, TypeScript, AWS（アイコンで表現）

【補足指示】
- 名刺のようなプロフェッショナル感
- 情報が整理されて見やすい
- シェアしたくなるデザイン
```

## 例6: プロフィールアイコン（マスコットキャラ）

### プロンプト

```
【用途】プロフィールアイコン (400x400、正方形)
【メインモチーフ】猫のマスコットキャラクター、エンジニア風
【スタイル】イラスト風、かわいい系
【カラースキーム】オレンジ（猫）、青（アクセント）、白背景
【背景】シンプルな白、軽いグラデーション

【構成】
- オレンジ色の猫キャラクター
- 丸メガネをかけている
- 小さなラップトップを抱えている
- にっこり笑顔
- 正面向き、顔を大きく

【補足指示】
- かわいくて親しみやすい
- 猫の特徴（耳、ひげ）を強調
- エンジニア要素（メガネ、PC）を取り入れ
- 小さくても猫と分かるデザイン
```
