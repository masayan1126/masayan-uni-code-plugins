# SNS画像フォーマット定義

## 用途別サイズ

| 用途 | サイズ | アスペクト比 | 特徴 |
|------|-------|------------|------|
| プロフィールアイコン | 400x400 | 1:1 | 円形にトリミングされることが多い |
| Xヘッダー | 1500x500 | 3:1 | 横長、プロフィール部分を避ける |
| X投稿用 | 1200x675 | 16:9 | 単独投稿用 |
| Instagram投稿 | 1080x1080 | 1:1 | 正方形 |
| Instagram ストーリー | 1080x1920 | 9:16 | 縦長 |

---

## スタイル定義

### 1. イラスト風
| 項目 | 設定 |
|------|-----|
| 特徴 | アニメ・漫画調、目が大きめ |
| 線 | はっきりした輪郭線 |
| 色 | ビビッドカラー、セルシェーディング |
| 適した用途 | クリエイター、エンタメ系 |

### 2. フラットデザイン
| 項目 | 設定 |
|------|-----|
| 特徴 | シンプル、モダン、ミニマル |
| 線 | 輪郭線なし、または細い線 |
| 色 | 限られた色数、フラットカラー |
| 適した用途 | ビジネス、テック系、ブランド |

### 3. 3D風
| 項目 | 設定 |
|------|-----|
| 特徴 | 立体的、グラデーション、質感 |
| 線 | 柔らかい影 |
| 色 | グラデーション、反射 |
| 適した用途 | ゲーム、メタバース、モダン |

### 4. 手書き風
| 項目 | 設定 |
|------|-----|
| 特徴 | 温かみ、手描きのタッチ |
| 線 | 不均一な線、スケッチ風 |
| 色 | 水彩風、パステル |
| 適した用途 | 親しみやすさ重視、個人ブランド |

## プロフィールアイコンの要素

### 推奨モチーフ
| カテゴリ | 例 |
|---------|-----|
| 人物 | 似顔絵、アバター、シルエット |
| 動物 | 猫、犬、鳥、架空の生き物 |
| アイテム | ロゴ、シンボル、道具 |
| 抽象 | 幾何学模様、イニシャル |

### 配色のポイント
- ブランドカラーを活用
- コントラストを確保
- 背景色は認識しやすく
- 派手すぎず、地味すぎず

### サイズ考慮事項
- 小さく表示されても認識できる
- 細かいディテールは避ける
- 中央にメインモチーフを配置
- 円形トリミングを考慮

---

## ヘッダー画像の要素

### レイアウトのポイント
- 左下のプロフィール部分を避ける
- 重要な要素は中央〜右側に配置
- 横長を活かしたパノラマ的構図

### 適したビジュアル
- 風景・背景
- パターン・テクスチャ
- ブランドビジュアル
- 作品ギャラリー

### テキスト使用時の注意
- 読みやすいサイズ
- 背景とのコントラスト
- プロフィール画像と被らない

---

## 投稿用画像の要素

### 目的別デザイン
| 目的 | ビジュアル | 配色 |
|------|----------|------|
| 告知 | テキスト中心 | 注目を集める色 |
| 作品紹介 | 作品がメイン | 作品を引き立てる |
| 日常 | カジュアル | 柔らかい色 |
| プロモーション | CTA重視 | ブランドカラー |

---

## NG事項

### プロフィールアイコン
- 複雑すぎるデザイン
- 文字の使用（読めない）
- 顔が小さすぎる
- 背景と同化する

### ヘッダー
- 重要な要素が左下
- 暗すぎる画像
- 情報過多
- 低解像度

### 投稿用
- 読めないテキスト
- 用途不明なデザイン
- ブランドとの不一致
