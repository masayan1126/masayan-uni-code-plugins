# プロフィール・SNS画像プロンプトテンプレート

## 基本構造

```
【用途】{purpose} ({size})
【メインモチーフ】{main_motif}
【スタイル】{style}
【カラースキーム】{color_scheme}
【背景】{background}

【構成】
{composition}

【補足指示】
{additional_notes}
```

## 用途別テンプレート

### プロフィールアイコン

```
【用途】プロフィールアイコン (400x400、正方形)
【メインモチーフ】{main_motif}
【スタイル】{style}
【カラースキーム】{color_scheme}
【背景】{background}

【構成】
- メインモチーフを中央に大きく配置
- 円形トリミングを考慮した余白
- シンプルで認識しやすいデザイン

【補足指示】
- 小さく表示されても認識できるデザイン
- 細かいディテールは避ける
- コントラストを確保
{additional_notes}
```

### Xヘッダー

```
【用途】Xヘッダー (1500x500、横長)
【メインビジュアル】{main_visual}
【スタイル】{style}
【カラースキーム】{color_scheme}
【背景】{background}

【構成】
- 横長を活かしたパノラマ的構図
- 左下のプロフィール部分は空ける
- 重要な要素は中央〜右側に配置

【テキスト】（任意）
{text_content}

【補足指示】
- ブランドイメージに合ったデザイン
- プロフィール画像との調和
{additional_notes}
```

### X投稿用

【メインビジュアル】{main_visual}

## スタイル別デフォルト設定

### イラスト風

```
【スタイル】イラスト風
- アニメ・漫画調のタッチ
- はっきりした輪郭線
- ビビッドカラー、セルシェーディング
- 目は大きめ、表情豊か
```

### フラットデザイン

```
【スタイル】フラットデザイン
- シンプル、モダン、ミニマル
- 輪郭線なし、または細い線
- 限られた色数（3-4色）
- フラットカラー、グラデーションなし
```

### 3D風

```
【スタイル】3D風
- 立体的、奥行きのある表現
- 柔らかい影、ハイライト
- グラデーション、反射
- 質感のあるマテリアル
```

### 手書き風

```
【スタイル】手書き風
- 温かみのあるタッチ
- 不均一な線、スケッチ風
- 水彩風、パステル調の色
- 手描きの質感
```
