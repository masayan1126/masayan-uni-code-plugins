# サンプルプロンプト集

## 例1: テキスト中心型「プログラミング学習法」

### プロンプト

```
【形式】YouTubeサムネイル (1280x720)
【タイトルテキスト】「独学3ヶ月でエンジニア転職」（画像中央に大きく配置）
【メインビジュアル】テキストが主役
【背景】濃紺から紫へのグラデーション背景、シンプル
【配色】白テキストと黄色アクセントの高コントラスト
【スタイル】テキスト中心型

【レイアウト】
- 中央にメインタイトル「独学3ヶ月でエンジニア転職」
- 下部にサブタイトル「〜完全ロードマップ公開〜」
- 右下にコードのアイコン（装飾）

【テキスト配置】
- メインタイトル: 画像中央、太いゴシック体、白色
- 「3ヶ月」の部分は黄色でハイライト
- サブタイトル: メインの下、やや小さめ、白色
- テキストに黒い影を追加

【補足指示】
- クリック率を意識したデザイン
- テキストは読みやすく大きめに
- 余白を十分に確保
- 日本語テキスト
- テック感のある雰囲気
```

## 例2: 人物・キャラ型「React入門」

### プロンプト

```
【形式】YouTubeサムネイル (1280x720)
【タイトルテキスト】「React入門」「初心者向け完全ガイド」
【メインビジュアル】若い男性エンジニア、ノートPCを持って笑顔、カジュアルな服装
【背景】Reactのロゴカラー（水色）のグラデーション
【配色】水色、白、ダークブルー
【スタイル】人物・キャラ型

【レイアウト】
- 右側にエンジニアキャラクター（60%）
- 左側にタイトルテキスト（40%）
- キャラクターの視線がテキストに向く
- 背景にReactのロゴを薄く配置

【テキスト配置】
- メインタイトル「React入門」: 左上、太いゴシック体、白色
- サブタイトル「初心者向け完全ガイド」: メインの下、やや小さめ
- Reactのロゴアイコンをタイトル横に

【補足指示】
- クリック率を意識したデザイン
- キャラクターは親しみやすい表情
- テック感とフレンドリーさのバランス
- 日本語テキスト
```

## 例4: 比較型「Mac vs Windows」

### プロンプト

```
【形式】YouTubeサムネイル (1280x720)
【タイトルテキスト】「エンジニアはどっちを選ぶべき？」
【メインビジュアル】左右対比のレイアウト、MacBookとWindowsノート
【背景】左側グレー（Mac風）、右側ブルー（Windows風）
【配色】左（シルバー、黒）、右（青、白）で対比
【スタイル】比較型

【レイアウト】
- 左側: Appleロゴ風のシルエット、MacBook
- 右側: Windowsロゴ風のシルエット、Windowsノート
- 中央に「VS」を大きく配置（赤色）
- 上部に質問形式のタイトル

【テキスト配置】
- タイトル「エンジニアはどっちを選ぶべき？」: 画像上部中央、白
- 「Mac」: 左側下部
- 「Windows」: 右側下部
- 「VS」: 中央に大きく、赤色、炎のエフェクト

【補足指示】
- クリック率を意識したデザイン
- 対決感を演出
- どちらかに偏らない公平なデザイン
- 日本語テキスト
- 緊張感のある雰囲気
```

## 例5: テキスト中心型「エラー解決」

### プロンプト

```
【形式】ブログサムネイル (1200x630)
【タイトルテキスト】「このエラー、5分で解決」
【メインビジュアル】テキストが主役、エラーメッセージ風の装飾
【背景】黒背景、ターミナル風のデザイン
【配色】赤（エラー）、緑（解決）、白テキスト
【スタイル】テキスト中心型

【レイアウト】
- 上部に赤いエラーメッセージ風のデザイン
- 中央にメインタイトル「このエラー、5分で解決」
- 「5分で解決」の部分は緑色でポジティブに
- ターミナルのカーソルアニメーション風

【テキスト配置】
- エラー表示: 上部、赤色、モノスペースフォント
- メインタイトル: 中央、白色、太字
- 「5分で」: 緑色ハイライト
- コード風のフォントで技術感を演出

【補足指示】
- クリック率を意識したデザイン
- 困っている人が「これだ！」と思うデザイン
- 解決への希望を感じさせる
- 日本語テキスト
```
