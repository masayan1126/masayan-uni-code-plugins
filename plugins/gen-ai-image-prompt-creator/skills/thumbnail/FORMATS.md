# サムネイルフォーマット定義

## プラットフォーム別サイズ

| プラットフォーム | サイズ | アスペクト比 | 用途 |
|----------------|-------|------------|------|
| YouTube | 1280x720 | 16:9 | 動画サムネイル |
| ブログ | 1200x630 | 約1.9:1 | OGP画像、アイキャッチ |
| X投稿 | 1200x675 | 16:9 | ポスト用画像 |

---

## サムネイルスタイル

### 1. テキスト中心型
| 項目 | 設定 |
|------|-----|
| 特徴 | 大きな文字でメッセージを伝える |
| 適した内容 | ハウツー、リスト記事、重要なお知らせ |
| レイアウト | 中央にテキスト、背景はシンプル |
| 配色 | 高コントラスト、読みやすさ重視 |

### 2. 人物・キャラ型
| 項目 | 設定 |
|------|-----|
| 特徴 | メインビジュアル＋テキスト |
| 適した内容 | インタビュー、レビュー、体験談 |
| レイアウト | 左または右に人物、反対側にテキスト |
| 配色 | 人物を引き立てる背景色 |

### 3. コンセプト型
| 項目 | 設定 |
|------|-----|
| 特徴 | 抽象的なビジュアルで興味を引く |
| 適した内容 | 技術解説、コンセプト紹介、エッセイ |
| レイアウト | グラフィカルな要素、アイコン中心 |
| 配色 | テーマに合わせたブランドカラー |

### 4. 比較型
| 項目 | 設定 |
|------|-----|
| 特徴 | Before/After、A vs Bの対比 |
| 適した内容 | 比較記事、変化の紹介、対決もの |
| レイアウト | 左右分割、対比を明確に |
| 配色 | 2色で対比を表現 |

---

## 必須要素チェックリスト

- [ ] キャッチーなタイトル（20文字以内）
- [ ] 高コントラストな配色
- [ ] 視認性の高いフォント
- [ ] 明確な焦点ポイント
- [ ] モバイルでも読めるサイズ

---

## 配色パターン

### 注目を集める配色
| パターン | メイン | アクセント | 背景 |
|---------|-------|----------|------|
| 情熱系 | 赤 | 白 | 黒 |
| 信頼系 | 青 | 白 | 濃紺 |
| 成長系 | 緑 | 白 | ダークグリーン |
| 警告系 | 黄 | 黒 | オレンジ |
| クール系 | 紫 | ピンク | 黒 |

### 読みやすさの原則
- 背景とテキストのコントラスト比は4.5:1以上
- テキストに影やアウトラインを追加
- 複雑な背景にはテキスト背景を敷く

---

## フォント使用ガイド

### 推奨フォントスタイル
- **太いゴシック体**: インパクト重視
- **丸ゴシック**: 親しみやすさ
- **明朝体**: 高級感、専門性

### テキストサイズ目安
- メインタイトル: 画像高さの15-20%
- サブタイトル: メインの50%程度
- 装飾テキスト: 小さめでアクセント

---

## NG事項

- テキストが小さすぎて読めない
- 情報が詰め込みすぎ
- 背景とテキストのコントラスト不足
- フォントの使いすぎ（2種類まで）
- 画像が暗すぎる
