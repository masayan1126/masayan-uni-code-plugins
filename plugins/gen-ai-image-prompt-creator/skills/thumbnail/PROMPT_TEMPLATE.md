## スタイル別テンプレート

### テキスト中心型

```
【形式】{platform}サムネイル ({size})
【タイトルテキスト】{title}（画像中央に大きく配置）
【メインビジュアル】テキストが主役
【背景】{background_color}のグラデーション背景、シンプル
【配色】{main_color}と{accent_color}の高コントラスト
【スタイル】テキスト中心型

【レイアウト】
- 中央にメインタイトル
- 上部または下部にサブタイトル
- 装飾は最小限

【テキスト配置】
- メインタイトル: 画像中央、太いゴシック体
- サブタイトル: メインの下、やや小さめ
- テキストに影またはアウトライン

【補足指示】
- クリック率を意識したデザイン
- テキストは読みやすく大きめに
- 余白を十分に確保
- 日本語テキスト
```

### 人物・キャラ型

> **注意**: 既存キャラクター使用時は、画像生成時にベースイラストを添付してください。

#### 既存キャラクター使用時
```
【形式】{platform}サムネイル ({size})
【タイトルテキスト】{title}
【メインビジュアル】添付画像のキャラクター
【背景】白（単色、シンプル）
【配色】白背景、黒テキスト、キャラクターの色はそのまま活かす
【スタイル】人物・キャラ型

【キャラクター指示】
- 添付画像のキャラクターをそのまま使用
- 画風・タッチ・色を維持
- ポーズは元画像そのままではなく、以下からランダムに選んで変更すること
  - 寝る、走る、頑張る、飛ぶ、びっくりする、喜ぶ、考える、手を振る など

【レイアウト】
- 下部中央にキャラクター
  - キャラクターはサムネの中央（左右の軸）に配置すること
- 上部から中部中央にタイトルテキスト
- 装飾は最小限、クリーンでシンプルに

【テキスト配置】
- メインタイトル: 上部中央、太いゴシック体、黒色、大きめ
- メインタイトルの下に手書き風のラフなアンダーライン（マーカーで引いたような太めのライン、アクセントカラーで強調）
- サブタイトル: メインの下、やや小さめ、黒色
- サブタイトルとメインの間にほんの少しだけ余白を入れること
- シンプルで読みやすいフォント

【補足指示】
- クリック率を意識したデザイン
- 添付キャラクターの画風・色を忠実に再現
- シンプルでクリーンなデザイン
- 日本語テキスト
- 余計な装飾は入れない
- テキストやキャラ画像の配置、整列、文字サイズ、バランスに気を使い、均等のとれた配置にすること
```

#### 新規キャラクター作成時
```
【形式】{platform}サムネイル ({size})
【タイトルテキスト】{title}
【メインビジュアル】{character_description}
【背景】{background}、キャラを引き立てる配色
【配色】{color_scheme}
【スタイル】人物・キャラ型

【レイアウト】
- 左側または右側にキャラクター（60%）
- 反対側にテキスト（40%）
- キャラクターの視線がテキストに向く

【テキスト配置】
- タイトル: キャラの反対側、太字
- サブテキスト: タイトルの下
- テキスト背景を敷いて視認性確保

【補足指示】
- クリック率を意識したデザイン
- キャラクターの表情を魅力的に
- テキストとキャラのバランス重視
- 日本語テキスト
```

### コンセプト型

```
【形式】{platform}サムネイル ({size})
【タイトルテキスト】{title}
【メインビジュアル】{concept_visual}（抽象的なグラフィック）
【背景】{background}
【配色】{color_scheme}、ブランドカラー活用
【スタイル】コンセプト型

【レイアウト】
- グラフィカルな要素を中心に配置
- タイトルは上部または下部
- アイコンやシンボルを効果的に使用

【テキスト配置】
- タイトル: 画像上部または下部
- グラフィックを邪魔しない位置
- 読みやすいフォントサイズ

【補足指示】
- クリック率を意識したデザイン
- 抽象的でも内容が伝わる工夫
- モダンでクリーンなデザイン
- 日本語テキスト
```

### 比較型

```
【形式】{platform}サムネイル ({size})
【タイトルテキスト】{title}（{item_a} vs {item_b}）
【メインビジュアル】左右対比のレイアウト
【背景】左右で色分け
【配色】{color_a}（左）と{color_b}（右）で対比
【スタイル】比較型

【レイアウト】
- 左側: {item_a}のビジュアル
- 右側: {item_b}のビジュアル
- 中央に「VS」または対比を示す要素
- 上部にタイトル

【テキスト配置】
- タイトル: 画像上部中央
- 各項目名: それぞれの側に配置
- VSマーク: 中央に大きく

【補足指示】
- クリック率を意識したデザイン
- 対比が一目で分かるデザイン
- 両者の違いを明確に表現
- 日本語テキスト
```

## プラットフォーム別の注意点

### YouTube
- モバイル表示を意識（小さくても読める）
- 右下にタイムスタンプが表示されるので避ける
- 再生ボタンが中央に来るので配慮

### ブログ
- SNSシェア時のプレビューを意識
- OGP画像として使用されることを考慮
- 両端がトリミングされる可能性を考慮
