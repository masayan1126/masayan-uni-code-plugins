# PIXTAメタデータ生成

## ワークフロー

1. **プロンプト受け取り**: 画像生成に使用したプロンプトを受け取る
2. **要素抽出**: プロンプトから被写体・スタイル・ポーズ・表情・背景を特定
3. **タイトル生成**: TITLE_GUIDE.md に従い3案作成（15〜30文字）
4. **タグ生成**: TAG_CATEGORIES.md の6カテゴリに沿って単語を抽出
5. **出力提示**: コピペしやすい形式で提示

## 参照ファイル

- [TITLE_GUIDE.md](TITLE_GUIDE.md): タイトル作成ルール
- [TAG_CATEGORIES.md](TAG_CATEGORIES.md): タグカテゴリ定義
- [EXAMPLES.md](EXAMPLES.md): 入出力サンプル

## 必須ルール

- タイトルは文章形式（単語羅列NG）
- タグは単語のみ（文章NG）
- プロンプト内容に忠実に（誤情報は損害賠償リスク）
- 類義語・上位概念も含める
