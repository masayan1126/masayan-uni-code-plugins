# PIXTAタイトル作成ガイド

## 基本ルール

| 項目 | 要件 |
|------|------|
| 文字数 | 15〜30文字程度 |
| 形式 | 文章（「〜する」「〜の」で繋がる） |
| 内容 | どのような作品かを簡潔に説明 |

## 良いタイトルの特徴

1. **具体的**: 被写体が明確にわかる
2. **簡潔**: 適度な長さ（15〜30文字）
3. **文章形式**: 単語同士の関係が明確

## 良い例

| タイトル | 文字数 | ポイント |
|---------|-------|----------|
| 芝生でボール遊びをするダルメシアンの子犬 | 19文字 | 動作・場所・被写体が明確 |
| 投げたボールを取ってくるダルメシアンの子犬 | 20文字 | 状況が具体的にわかる |
| クレヨンで描いたシロハラインコのイラスト | 19文字 | スタイル・被写体が明確 |
| 元気に跳ねるシロハラインコ | 12文字 | 動作・被写体が簡潔 |

## 悪い例

### 短すぎる（単語の羅列）

```
ダルメシアン 芝生
インコ イラスト
```
→ 単語の関係性が不明確

### 長すぎる

```
5月20日の午後3時の××公園の芝生と、全力で投げたスイカ柄のボールを喜んで取ってくるダルメシアンの子犬
```
→ 情報過多で分かりづらい

### キーワード乱用

```
子犬 赤ちゃん 動物 犬 ダルメシアン 白黒 走る 芝生 わんこ
```
→ 文章になっていない、主役が不明

## タイトル生成のコツ

### 1. 被写体を明確に
- 「インコ」→「シロハラインコ」
- 「鳥」→「跳ねるシロハラインコ」

### 2. 特徴を加える
- スタイル: 「手描きの〜」「クレヨン風の〜」
- 動作: 「跳ねる〜」「首をかしげた〜」
- 雰囲気: 「元気いっぱいの〜」「愛らしい〜」

### 3. 文章として自然に
- 「○○する○○」
- 「○○な○○」
- 「○○風の○○」

## イラスト向けタイトルパターン

| パターン | 例 |
|---------|-----|
| 【スタイル】で描いた【被写体】 | クレヨンで描いたシロハラインコ |
| 【動作】する【被写体】の【スタイル】イラスト | 跳ねるインコの手描きイラスト |
| 【雰囲気】な【被写体】 | 元気いっぱいのシロハラインコ |
| 【スタイル】風【被写体】イラスト | 色鉛筆風シロハラインコイラスト |
