# PIXTAタグカテゴリ

## 6つのカテゴリ

| カテゴリ | 説明 | 入れるべき情報 |
|---------|------|--------------|
| 被写体 | メインの対象物 | 名称、種類、分類 |
| テーマ | 用途・ジャンル | 使用シーン、カテゴリ |
| 状態・動作・色 | 動き・特徴・色 | 動詞、形容詞、色名 |
| 表情・感情・雰囲気 | 印象・ムード | 感情、雰囲気 |
| 写り方 | 構図・技法 | 撮影/描画スタイル |
| 背景 | 背景情報 | 場所、背景色 |

---

## 1. 被写体

メインとなる対象物の名称、類義語、上位概念を含める。

### 例: 動物系
| 具体 → 抽象 |
|------------|
| シロハラインコ → インコ → 鳥 → 小鳥 → ペット → 動物 |
| ダルメシアン → 犬 → 子犬 → ペット → 動物 |

### 例: 人物系
| 具体 → 抽象 |
|------------|
| ビジネスマン → 男性 → 人物 → 日本人 → 20代 |
| 親子 → 家族 → 人物 → 複数 |

### 例: 物品系
| 具体 → 抽象 |
|------------|
| りんご → 果物 → 食べ物 → 食材 |
| パソコン → PC → デバイス → 電子機器 |

---

## 2. テーマ

作品の用途やジャンルを表すタグ。

| カテゴリ | タグ例 |
|---------|-------|
| ビジネス | ビジネス, 仕事, オフィス, 会議 |
| ライフスタイル | ライフスタイル, 日常, 生活 |
| ペット・動物 | ペット, 動物, かわいい, 癒し |
| 料理・食べ物 | 料理, グルメ, 食材, レシピ |
| 風景・自然 | 風景, 自然, 景色, 観光 |
| アート・イラスト | イラスト, アート, 絵, 作品 |
| 季節イベント | 春, 夏, 秋, 冬, クリスマス, 正月 |

---

## 3. 状態・動作・色

被写体の状態や動作、特徴的な色を表すタグ。

### 動作
| 分類 | タグ例 |
|-----|-------|
| 移動系 | 走る, 歩く, 跳ねる, 飛ぶ, ジャンプ |
| ポーズ系 | 立つ, 座る, 寝転ぶ, 首をかしげる |
| 動物系 | ホッピング, 羽ばたく, 毛づくろい |

### 色
| 分類 | タグ例 |
|-----|-------|
| 基本色 | 赤, 青, 緑, 黄, 白, 黒, オレンジ, ピンク |
| 特徴色 | パステル, 鮮やか, 淡い, カラフル |

作品の印象やムードを表すタグ。

| 分類 | タグ例 |

| ポジティブ | 楽しい, 嬉しい, 元気, 明るい, 爽やか |
| 穏やか | 癒し, リラックス, 穏やか, 優しい, 温かい |

### 構図・アングル

|-----|------|

| 全身 | 全体が見える構図 |
| 俯瞰 | 上から見下ろす |
| 正面 | 正面からの視点 |
| 横顔 | 横からの視点 |

### スタイル・技法
| タグ | 説明 |
|-----|------|

## 6. 背景

背景に関する情報を表すタグ。

| 分類 | タグ例 |
|-----|-------|
| 色 | 白バック, 黒バック, クリーム色, パステル |
| 場所 | 屋内, 屋外, オフィス, 公園, 自然 |
| 状態 | シンプル背景, 無地, コピースペース |
| 素材感 | 画用紙, テクスチャ, グラデーション |

## タグ付けの注意点

### やるべきこと
- 類義語も含める（インコ→鳥、小鳥）
- 上位概念も含める（シロハラインコ→インコ→ペット→動物）
- 特徴的な色は入れる
- 人物なしなら「人物なし」を入れる

### やってはいけないこと
- 作品に含まれない情報を入れる
- 文章形式で入れる（単語のみ）
- 曖昧・誤った情報を入れる
- 関係ない季節タグを入れる（桜に「クリスマス」など）
