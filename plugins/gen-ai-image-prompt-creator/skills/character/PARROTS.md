# インコモチーフ定義

スケッチ調でインコを描く際の種類別特徴と描き分けのポイント。

---

## 対応種類

### 1. シロハラインコ (White-bellied Caique)

| 項目 | 特徴 |
|------|-----|
| 体型 | ずんぐり丸い、中型（約23cm） |
| 頭部 | オレンジ〜アプリコット色 |
| 胸部 | 純白（最大の識別ポイント） |
| 翼・背 | 鮮やかな緑 |
| 尾 | 短め、緑 |
| くちばし | 肌色〜ピンク |
| 足 | ピンク |
| 性格表現 | 活発、好奇心旺盛、いたずら好き、陽気 |
| 特徴的ポーズ | ホッピング（跳ねる）、逆さまぶら下がり、ゴロン |

### 2. コザクラインコ (Peach-faced Lovebird)

| 項目 | 特徴 |
|------|-----|
| 体型 | 小型（約15cm）、ずんぐりコンパクト |
| 顔 | ピーチピンク〜オレンジ（ピーチフェイス） |
| 額 | 赤〜オレンジ |
| 体 | 明るい緑 |
| 腰 | 青色のアクセント |
| 尾 | 短い、緑に青や赤のアクセント |
| くちばし | 明るいオレンジ〜赤 |
| 足 | グレー |
| 性格表現 | 甘えん坊、情熱的、やきもち焼き、寄り添い好き |
| 特徴的ポーズ | パートナーに寄り添う、首をかしげる、羽繕い |

### 3. サザナミインコ (Lineolated Parakeet / Linnie)

| 項目 | 特徴 |
|------|-----|
| 体型 | 小型（約16cm）、細身でスマート |
| 全体色 | 緑（ノーマル）、青、ルチノー、クリームなど品種多様 |
| 模様 | 波状の細かい縞模様（バーリング）が背中と翼に |
| 尾 | 短い、先端が尖る |
| くちばし | 肌色〜淡いピンク |
| 足 | 肌色 |
| 性格表現 | おっとり、穏やか、静か、マイペース |
| 特徴的ポーズ | 水平姿勢、枝に沿って横歩き、じっと観察 |

### 4. オカメインコ (Cockatiel)

| 項目 | 特徴 |
|------|-----|
| 体型 | 中型（約30cm）、スリムで優雅 |
| 冠羽 | 特徴的な冠羽（感情で立てたり寝かせたり） |
| 頬斑 | 鮮やかなオレンジ（チークパッチ） |
| 体色（ノーマル） | グレー基調、翼に白いウィングバー |
| 体色（ルチノー） | 黄色〜クリーム、白、オレンジ頬斑 |
| 尾 | 長い |
| くちばし | グレー |
| 足 | グレー |
| 性格表現 | 優しい、歌好き、甘えん坊、おしゃべり |
| 特徴的ポーズ | 冠羽を立てる、歌うポーズ、頭を下げて撫でて要求 |

---

## 描き分けのポイント

| インコ | 最大の識別ポイント | シルエット特徴 |
|--------|------------------|---------------|
| シロハラインコ | 白い胸 + オレンジ頭 | 丸くてずんぐり |
| コザクラインコ | ピーチフェイス + 赤いくちばし | 小さくてコンパクト |
| サザナミインコ | 波状の縞模様 + 水平姿勢 | 細身でスマート |
| オカメインコ | 冠羽 + オレンジ頬斑 | スリムで長い尾 |

---

## スケッチ表現別のポイント

### 鉛筆・デッサン風
- 羽毛の質感をハッチングで表現
- 頬斑や模様は濃淡で描き分け
- 目の輝きをハイライトで強調

### 水彩スケッチ風
- 羽の色はにじみで柔らかく
- ピーチフェイスやオレンジ頬斑はウェットオンウェット
- 白い部分は紙の白を活かす

### ボールペン/万年筆風
- 羽毛を細かい線の集合で表現
- 模様はクロスハッチングで陰影
- くちばしと目は繊細な線で

### クレヨン/色鉛筆風
- 羽の重なりをストロークで表現
- 柔らかい発色で温かみを出す
- ざらつき質感で親しみやすさを

## よく使う表現

### 感情表現
| 感情 | インコの表現 |
|------|------------|
| 嬉しい | 冠羽を立てる（オカメ）、羽を膨らませる |
| リラックス | 片足立ち、目を細める |
| 興奮 | 羽を広げる、くちばしを開ける |
| 甘える | 頭を下げる、寄り添う |
| 好奇心 | 首をかしげる、目を見開く |

### 動作・ポーズ
| ポーズ | 説明 |
|-------|-----|
| 止まり木 | 横向き、自然な姿勢 |
| 正面顔 | 首をかしげて可愛らしく |
| 飛翔 | 翼を広げて羽ばたく |
| 食事 | くちばしに餌をくわえる |
| 毛繕い | 羽を整える仕草 |
| 睡眠 | 羽に顔をうずめる、片足立ち |
