## バリエーション用テンプレート

### 表情差分

```

【表情バリエーション】
1. 笑顔（にっこり）
2. 怒り（眉をひそめる）
3. 悲しみ（涙目）
4. 驚き（目を見開く）
5. 照れ（頬を赤らめる）

- 同一キャラクターの表情差分
- ポーズと服装は統一
- 表情以外の要素は変えない
```
