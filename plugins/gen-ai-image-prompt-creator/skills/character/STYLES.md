### 7. スケッチ調

スケッチ調には4つのサブスタイルがあります。

#### 7-1. 鉛筆・デッサン風
| 項目 | 設定 |
|------|-----|
| 特徴 | クラシックなスケッチ、学術的・アーティスティックな印象 |
| 線 | グラファイト鉛筆の濃淡、2B〜6Bの太さ変化 |
| 塗り | ハッチング、クロスハッチングで陰影表現 |
| 色 | モノクロ基本、セピア調も可 |
| 質感 | 紙の質感、鉛筆のざらつき |
| 適した用途 | アート作品、観察スケッチ、教育素材、額装向け |

#### 7-2. 水彩スケッチ風
| 項目 | 設定 |
|------|-----|
| 特徴 | 柔らかく透明感のある表現、癒し系 |
| 線 | 薄い鉛筆の下書きライン、または線なし |
| 塗り | 水彩のにじみ、ウェットオンウェット、色の重なり |
| 色 | パステル調、透明感のある淡い色 |
| 質感 | 水彩紙のテクスチャ、絵の具の溜まり |
| 適した用途 | 絵葉書、ナチュラル系グッズ、癒し系アイコン、インテリア |

#### 7-3. ボールペン/万年筆風
| 項目 | 設定 |
|------|-----|
| 特徴 | 繊細な線画、クラシカルで知的な印象 |
| 線 | 一定の太さ、インクの濃淡、クロスハッチング |
| 塗り | 線のみで構成、塗りなし or 点描（スティップリング） |
| 色 | 黒、ブルーブラック、セピア |
| 質感 | 滑らかなペン運び、インクのにじみ |
| 適した用途 | 手帳イラスト、文房具グッズ、スタンプ、レターヘッド |

#### 7-4. クレヨン/色鉛筆風
| 項目 | 設定 |
|------|-----|
| 特徴 | 温かみのある手描き感、やわらかく親しみやすい印象 |
| 線 | ざらついた輪郭、不均一で味のある線 |
| 塗り | 重ね塗り、ストローク跡が残る、はみ出し許容 |
| 色 | 鮮やかだが柔らかい発色 |
| 質感 | 画用紙、クレヨンのワックス感、色鉛筆の木目 |
| 適した用途 | 子供向け、絵本風、温かみのあるグッズ、ポストカード |

### 8. ゆるキャラ調

動物をモチーフにした、かわいいマスコットキャラクター向けスタイル。

| 項目 | 設定 |
|------|-----|
| 特徴 | ゆるい動物マスコット、丸っこい体型、短い手足 |
| 等身 | 2〜3等身 |
| 線 | 太めの濃い茶色アウトライン、少しだけ手描きの揺れ |
| 塗り | フラット塗り、影は最小限 |
| 色 | 低彩度パステル配色、5色以内 |
| 顔 | 点目・小さい口・ほっぺチーク |
| 効果 | シンプルなワンポイント（キラッ、汗、Zzz） |
| 適した用途 | LINEスタンプ、SNSアイコン、ステッカー、グッズ |

#### ゆるキャラ調の目的別パターン

##### A. 丸背景アイコン版
SNSアイコンやプロフィール画像向け。丸い背景の上にキャラを配置。
- 構図: 1:1、中央配置、安全余白あり
- 背景: パステル色の円形 + 白い外周

##### B. ステッカー（透過前提）版
LINEスタンプやステッカー素材向け。
- 構図: 1:1、中央配置、切り抜きしやすい輪郭
- 背景: 単色または透明

##### C. 2×2の4体セット版
一覧画像やバリエーション表示向け。
- 構図: 2×2レイアウト、ポーズ違い4体
- 背景: パステル単色
- 効果: 各キャラに小さな手描き記号

##### D. キャラデザイン案出し（表情差分）版
キャラクターデザイン資料向け。
- 構図: 同一キャラ×3パターン、整列
- 背景: 白、余白多め
- 表情: 通常/にっこり/びっくり等

#### ゆるキャラ調のスタイル固定プロンプト

**STYLE FIX（毎回先頭に入れる）**
```
ゆるい動物のマスコットキャラクター、2〜3頭身、丸っこい体型、短い手足、表情はシンプル（点目・小さい口）、ほっぺに薄いチーク、太めの濃い茶色アウトライン、少しだけ手描きの線の揺れ、フラット塗り、影は最小限、低彩度パステル配色、余白多め、クリーンで可愛いアイコンデザイン、シンプルなワンポイント（キラッ、汗、Zzzなど）
high quality cute mascot illustration, thick dark-brown outline, minimal shading, flat pastel colors, simple face, clean negative space
```

**NEGATIVE（崩れ防止・毎回末尾に入れる）**
```
写実的、3D、CG、厚塗り、強いグラデーション、ハイライト過多、細かい毛並み、複雑な背景、文字、ロゴ、透かし、過度なディテール、リアルな質感、フォト、ピクセルアート、AIっぽいテカり、過剰な陰影
```
