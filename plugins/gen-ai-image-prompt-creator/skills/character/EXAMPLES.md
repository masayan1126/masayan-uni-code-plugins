## 例1: アニメ調エンジニアキャラ

### プロンプト

```
【画風】アニメ調
- 日本アニメ風のタッチ
- 目が大きめ、輝く瞳
- はっきりした輪郭線
- セルシェーディング
- 6等身

【キャラクター名】テクノ

【外見設定】
- 性別・年齢: 男性、20代後半
- 髪: 黒髪ショート、少し寝癖がある、前髪が目にかかる
- 目: 青色、知的な印象、丸メガネをかけている
- 服装: 紺色のパーカー、白いTシャツ、ジーンズ
- 体型: 細身、普通体型

【ポーズ】片手でノートPCを持ち、もう片手でコーヒーカップを持つ
【表情】自信のある笑顔、少し得意げ
【背景】シンプルな白背景、コードの断片が薄く浮かぶ

【アクセサリー・小物】
- 丸メガネ
- ワイヤレスイヤホン

- コーヒーカップ

【補足指示】
```

## 例7: スケッチ調（水彩）- コザクラインコ

### プロンプト

```
【画風】スケッチ調 - 水彩スケッチ風
- 柔らかく透明感のある表現
- 薄い鉛筆の下書きライン、または線なし
- 水彩のにじみ、ウェットオンウェット
- パステル調、透明感のある淡い色
- 水彩紙のテクスチャ

【モチーフ】コザクラインコ（Lovebird）

【インコの特徴】
- 種類: コザクラインコ（Peach-faced Lovebird）
- 体色: ピーチピンクの顔、明るい緑の体、赤いくちばし（にじみで柔らかく表現）
- 識別ポイント: ピーチフェイス、小さくてずんぐり
- 表現ポイント: 顔のピンクを水彩のにじみで、緑の体との境界は自然なグラデーション

【ポーズ】首をかしげて見上げる、片足を上げて
【表情】目を細めて甘える表情

【構図】
- アスペクト比: 1:1（アイコン用）
- 背景: 淡いクリーム色、水彩のにじみ
- 視点: 正面やや上から

【スケッチ技法】
- 線: 薄い鉛筆線、一部消えかけの柔らかさ
- 塗り: ウェットオンウェット、ピンクと緑のグラデーション
- 質感: 水彩紙のざらつき、絵の具の溜まり

【補足指示】
- 癒し系SNSアイコンとして使用
- 優しく穏やかな雰囲気
- 水彩特有の透明感を活かす
```

## 例8: スケッチ調（鉛筆・デッサン）- オカメインコ

### プロンプト

```
【画風】スケッチ調 - 鉛筆・デッサン風
- クラシックなスケッチ、学術的な印象
- グラファイト鉛筆の濃淡、2B〜6Bの太さ変化
- ハッチング、クロスハッチングで陰影
- モノクロ基本
- 紙の質感、鉛筆のざらつき

【モチーフ】オカメインコ（Cockatiel）

【インコの特徴】
- 種類: オカメインコ（Cockatiel）
- 体色: モノクロの濃淡で表現（グレーの体、白い翼のバー、オレンジ頬斑は中間トーン）
- 識別ポイント: 特徴的な冠羽、長い尾
- 表現ポイント: 冠羽の立体感をハッチングで、頬斑は円形の柔らかい陰影

【ポーズ】冠羽を立てて歌うポーズ、くちばしを少し開けて
【表情】目を閉じて気持ちよさそう

【構図】
- アスペクト比: 3:4（縦長）
- 背景: クリーム色の紙、余白を活かす
- 視点: 横向き、頭部にフォーカス

【スケッチ技法】
- 線: グラファイト鉛筆、輪郭は太めの4B、羽毛は細めのHB
- 塗り: ハッチングで羽毛の流れを表現、目にハイライト
- 質感: 紙のざらつき、鉛筆の粒子感

【補足指示】
- 自然史イラストレーション風
- 鳥類図鑑に載っていそうな雰囲気
- 細部まで丁寧な観察スケッチ
```

## 例10: スケッチ調（ボールペン/万年筆）- サザナミインコ

### プロンプト

```
【画風】スケッチ調 - ボールペン/万年筆風
- 繊細な線画、クラシカルで知的な印象
- 一定の太さ、インクの濃淡
- 線のみで構成、クロスハッチング
- ブルーブラック
- 滑らかなペン運び

【モチーフ】サザナミインコ（Lineolated Parakeet）

【インコの特徴】
- 種類: サザナミインコ（Lineolated Parakeet / Linnie）
- 体色: 線の密度で表現（緑の濃淡、波状の縞模様）
- 識別ポイント: 波状の細かい縞模様（バーリング）、水平姿勢
- 表現ポイント: 縞模様を細かいハッチングで、スマートな体型を繊細な線で

【ポーズ】枝に止まって水平姿勢、じっとこちらを見つめる
【表情】穏やかな表情、目を細めてリラックス

【構図】
- アスペクト比: 4:3（横長）
- 背景: 白背景、枝のみシンプルに描写
- 視点: 真横

【スケッチ技法】
- 線: ブルーブラックのインク、均一な太さ、強弱で立体感
- 塗り: 塗りなし、縞模様はクロスハッチングで表現
- 質感: インクの滑らかさ、万年筆特有の線の美しさ

【補足指示】
- 手帳やノートに描いたような雰囲気
- おっとりした性格を静かな構図で表現
- 繊細で緻密な線画
```
