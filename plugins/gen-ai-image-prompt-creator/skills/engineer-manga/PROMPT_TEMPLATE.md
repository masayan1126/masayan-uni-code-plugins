## 品質向上のコツ

### キャラクター一貫性の確保（最重要）
- 画像AIにプロンプトを入力する際、必ずキャラクターの元画像を一緒に添付する
- プロンプト内では「添付画像のキャラクター」として参照する
- 以下の指示を含めることで一貫性を強化:
  ```
  添付画像を参照し、外観・プロポーション・配色を全コマで厳密に一貫させてください。
  ```

### キャラクター構成
添付画像には以下のキャラクターが含まれる：
- **男の子**: 主人公エンジニア
- **インコ（数羽）**: 相棒として一緒に登場

プロンプト生成時は、男の子とインコを両方とも漫画に登場させること。インコはリアクション役やマスコット的な役割で活用できる。

### シナリオの階層的記述
各コマは以下の4要素を明確に分けて記述する：
1. **場面設定**: どこで、いつ（例: 教室・昼休み）
2. **描写**: キャラクターの行動や状況
3. **セリフ**: 発言やモノローグ
4. **効果音**: オノマトペ

### ネガティブプロンプト（崩れ防止）
```
除外: 指が6本以上、不自然な体勢、コマごとの服装変化、顔が2つ
```

### カメラ距離の指定
- **ワイド（全体）**: 状況説明、複数キャラ登場
- **ミディアム**: 通常の会話シーン
- **クローズアップ**: 表情や決め場面
- **大アップ**: 衝撃的な瞬間

### 対話形式による修正（ダメ出しテクニック）
生成後に修正が必要な場合、具体的に指示する：
- 「2コマ目の視線がおかしい。AはカメラではなくBを見て」
- 「セリフの文字が潰れているので、フキダシを大きくして」
- 「3コマ目のインコの位置を主人公の肩に変更して」
