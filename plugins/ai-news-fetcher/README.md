# AI News Fetcher Plugin

Fetch the latest AI news from the web and generate bilingual markdown articles.

## Prerequisites

- **Tavily MCP**: This plugin requires Tavily MCP to be configured.
  - Get API key from https://tavily.com
  - Add to your MCP settings

## Features

- Fetch AI news using Tavily MCP
- Generate articles organized by date
- Bilingual output (Japanese/English)

## Usage

Use keywords like "AIニュース取得" or "fetch AI news".
