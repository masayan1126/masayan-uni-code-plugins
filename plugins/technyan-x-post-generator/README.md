# Tech-nyan X Post Generator Plugin

Generate X (Twitter) post drafts for Tech-nyan (テクにゃん.) character from articles.

## Features

- Generate 3 patterns each for Japanese and English
- Tech-nyan character voice and style (「〜にゃ」「〜にゃん」)
- Include article URL, hashtags, and OGP image instructions
- Template-based generation for technyanai.com

## Usage

```
/technyan-x-post src/content/articles/YYYYMMDD/article-slug/ja.md
```
